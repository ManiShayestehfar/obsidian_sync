>[!info] Definition
>A triplet $(\Omega, \mathcal{F}, P)$ is a **Probability Space** if  
>1. $\Omega$ is a [[Basic Definitions|sample space]]
>2. $\mathcal{F}$ is a [[Sigma-Algebra]] 
>3. $P$ is a [[Probability Measure]] on $(\Omega, \mathcal{F})$


## Example

### Rolling a fair die

- $\Omega = \{1,2,3,4,5,6\}$
- $\mathcal{F} = 2^{\Omega}$
- $P(\{i\}) = 1/6$  for $i = 1,...,6$
- $P(A) = |A| / 6$  for $A \in \mathcal{F}$
